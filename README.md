# CricketVision AI

An AI-powered Cricket Match Analysis System built using Computer Vision and Deep Learning.

## Features

- Cricket object detection using YOLOv8
- Ball detection
- Bat detection
- Player detection
- Stumps detection
- Video inference
- Object tracking (upcoming)
- Match analytics (upcoming)

## Tech Stack

- Python
- YOLOv8
- OpenCV
- PyTorch
- Roboflow
